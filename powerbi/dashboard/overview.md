# Executive Dashboard Overview

This folder contains visual references and documentation for the Tailwind Traders
Executive Dashboard deployed on **Power BI Service**.

The dashboard was created by pinning visuals from the Sales and Profit Overview
reports and configuring KPI tiles for rapid performance monitoring.
